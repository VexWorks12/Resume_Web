## About

This is my personal resume website showcasing my skills, education, certifications, and projects as an IT student and aspiring Front-End Developer. Built entirely without frameworks — just clean HTML, CSS, and JavaScript.

---

## Features

- Clean and readable resume layout
- Skills section with styled badges
- Education, Certifications, and Interests
- Experience section
- Contact information
- Fully responsive design
- No frameworks, pure HTML/CSS/JS
